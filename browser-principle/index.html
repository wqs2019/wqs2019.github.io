# 4. 浏览器中的页面循环系统 (6讲)
4-1. 消息队列和事件循环：页面是怎么活起来的 4-2. Webapi：setTimeout是怎么实现的 4-3. Webapi：XMLHttpRequest是怎么实现的 4-4. 宏任务和微任务：不是所有的任务都是一个待遇 4-5. 使用Promise告别回调函数 4-6. async await使用同步方式写异步代码
